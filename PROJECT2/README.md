Project Overview
This is Netflix Data Analysis project using Python, pandas, matplotlib, and seaborn.  
The project provides insights on Netflix content, including:
- Movies vs TV Shows distribution  
- Top countries producing Netflix content  
- Top directors by number of shows  
- Most popular genres  
- Release year trends  

How to Run
1. Make sure you have the required Python libraries:
pip install pandas matplotlib seaborn

The script will output:
Key insights printed in the terminal
Charts sequentially popping up showing:
1.Movies vs TV Shows distribution
2.Top 10 countries producing content
3.Top 10 directors
4.Top 10 genres
5.Number of shows added per release year
