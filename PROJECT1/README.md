## Project Overview
This project demonstrates a complete data analysis workflow using Python and pandas.  
The dataset used is `train.csv` and the analysis includes:
- Data cleaning (handling missing values and duplicates)  
- Exploratory data analysis (summary statistics)  
- Sales analysis by region  
- Data visualization using Matplotlib  
## Project Structure
1.Data-Analytics-Portfolio/ │ ├─ data/ │   └─ train.csv ├─ Scripts/ │   └─ analysis.py └─ README.md 
## How to Run
1. Clone the repository:  
git clone https://github.com/Sadaf008/Data-Analytics-Portfolio.git

2.Navigate to the folder:
cd Data-Analytics-Portfolio/Scripts

3.Install required packages (if not installed):
pip install pandas matplotlib

4.Run the analysis script:
python analysis.py
